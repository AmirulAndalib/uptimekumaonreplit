将以下代码粘贴至Replit Shell后回车

`git clone https://github.com/valetzx/uptimekumaonreplit && mv -b uptimekumaonreplit/* ./ && mv -b uptimekumaonreplit/.[^.]* ./ && rm -rf *~ && rm -rf uptimekumaonreplit`

当加载完 Loading Nix environment... 后点击绿色 ▶ Run
